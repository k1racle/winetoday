Виноделие сегодня

Разработка современного новостного портала о виноделии

Главное правило

Никаких заглушек, моков и временных решений. Полностью рабочий продукт. Все страницы, все ссылки, все интерфейсы и функции должны работать на 100%. После запуска docker compose up сайт должен быть полностью функциональным, красивым и наполненным реальным контентом.

Обязательное требование: Весь сайт и вся админ-панель — только на русском языке. Никаких других языков на данном этапе.

-----
Технологический стек

- Frontend: Next.js 16.2+ (App Router, TypeScript, Server Components)
- Backend / CMS: Strapi 5.42+
- База данных: PostgreSQL 17
- Кэш: Redis 7
- Поиск: Meilisearch
- Аутентификация: Auth.js (NextAuth v5)
- Графики и диаграммы: Recharts + Chart.js
- Стили: Tailwind CSS + CSS Variables
- Анимации: Framer Motion
- Drag & Drop: @dnd-kit
- PWA: Next-PWA
- Иконки: Lucide React
- Контейнеризация: Docker + Docker Compose + Nginx

Дизайн: Чёрно-зелёно-белая цветовая схема, Apple-like минимализм, чистая типографика, плавные анимации, тонкие границы и тени.

-----
Content Types в Strapi 5

Основные сущности:

- Article (лонгриды)
- News (новости)
- Event (мероприятия)
- Video (видео)
- Author (авторы)

Компоненты:

- TastingNote (дегустационные заметки с поддержкой radar-диаграмм)
- Dynamic Zones (для блочного построения статей и новостей)

Таксономия:

- Category, Tag, Region, Grape

Global Settings (Single Type) — все глобальные настройки сайта.

-----
Основной функционал

Дизайн и интерфейс:

- Полностью адаптивный mobile-first дизайн
- Поддержка светлой и тёмной темы
- Drag-and-drop конструктор хедера, футера, мобильного нижнего меню и сайдбара
- Бегущая строка с настройками
- Мобильный выдвижной сайдбар
- PWA (offline-режим, install prompt, push-уведомления)

Контент и страницы:

- Главная страница
- Ленты новостей, статей, видео, мероприятий
- Детальные страницы каждой сущности
- Page Builder на основе Dynamic Zones + Live Preview
- Календарь мероприятий в стиле Яндекс Афиша

Пользовательские функции:

- Полнотекстовый поиск
- Фильтры, теги, категории, бесконечная прокрутка
- Личный кабинет (регистрация, профиль, избранное, читать позже, история просмотров, подписки)
- Система комментариев с модерацией
- Избранное и сохранение контента
- Подписка на рассылки по категориям
- Рекомендации контента

Графики и визуализация:

- Радарные диаграммы для дегустационных заметок (ароматика, вкус, кислотность, танины и т.д.)
- Гистограммы, линейные графики, круговые диаграммы
- Сравнение вин
- Аналитические графики популярности контента
- Все графики адаптивны, с поддержкой тем, анимациями и русскими подписями

SEO и производительность:

- Автоматический sitemap, robots.txt, structured data (JSON-LD)
- Оптимизация изображений (WebP/AVIF)
- Core Web Vitals: LCP < 1.5s, FID < 50ms, CLS < 0.05
- Кэширование (Next.js + Redis)

Админ-панель:

- Кастомная чёрно-зелёная тема
- Drag-and-drop редактор хедера, футера и меню
- Панель управления темами, цветами, шрифтами
- SEO-превью
- Аналитический дашборд с графиками
- Управление пользователями, комментариями, рассылками

Этапы разработки

1. Настройка Docker Compose + базовая инфраструктура
1. Настройка Strapi 5 (Content Types, компоненты, Global Settings, русский интерфейс)
1. Seed данных (15–20 реальных материалов на русском)
1. Frontend: базовая структура, дизайн-система, темы
1. Page Builder + виджеты хедера/футера
1. Все функциональные модули (поиск, кабинет, комментарии, избранное и т.д.)
1. Модуль графиков и визуализации
1. PWA, SEO, производительность
1. Тестирование (Playwright) и финальная оптимизация
1. Подготовка production-сборки
